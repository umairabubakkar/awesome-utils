# Awesome Utils

A collection of useful utility functions for JavaScript/TypeScript projects.

## Installation

```bash
npm install awesome-utils
```

## Features

- String manipulation utilities
- Date formatting helpers
- Array transformation functions
- Object deep merge utilities

## Usage

```javascript
const { capitalize, formatDate, deepMerge } = require('awesome-utils');

console.log(capitalize('hello world')); // Hello World
console.log(formatDate(new Date())); // 2024-01-01
```
